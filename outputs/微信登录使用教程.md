# 微信登录接入文档

## 一、前置条件

在微信开发者工具中确认：
- [x] 项目已关联云开发环境（project.config.json 中 appid 正确）
- [x] 云开发已开通（开发者工具 → 云开发 → 开通）

---

## 二、操作步骤

### 步骤 1：上传 login 云函数

1. 在微信开发者工具中，展开 cloudfunctions/ 目录
2. 右键 login → 上传并部署云函数（选择「上传并部署：所有文件」）
3. 等待上传完成（约 10-30 秒）

### 步骤 2：创建 users 数据库集合

1. 在微信开发者工具中点击左上角 云开发 按钮
2. 进入 数据库 标签页
3. 点击 新建集合，名称填写 users（注意全小写）
4. 权限设置选择「仅创建者可读写」（默认安全规则即可）

### 步骤 3：创建云存储目录（用于保存头像）

1. 在云开发控制台 → 存储 标签页
2. 点击 新建文件夹，名称填写 avatars
3. 权限：默认（仅创建者可读写）

### 步骤 4：在开发者工具预览测试

1. 点击 编译 或 预览
2. 切换到「我的」Tab 页
3. 点击「微信一键登录」按钮
4. 观察控制台输出，确认无报错

## 三、登录流程说明

### 第一次使用

用户点击「微信一键登录」
  ↓
wx.login() 静默获取临时 code（无弹窗）
  ↓
调用 login 云函数
  ↓
云函数通过 cloud.getWXContext() 获取 OPENID
  ↓
查询 users 集合中是否有已保存的信息
  ↓
返回 { openid, nickName, avatarUrl } → 写入本地缓存
  ↓
显示资料编辑界面

### 编辑头像昵称（微信官方最新方案）

点击头像区域 → open-type="chooseAvatar" 触发
  ↓
弹出微信头像选择器 → 用户选择
  ↓
临时文件路径自动上传到云存储 avatars/
  ↓
调用云函数保存 avatarUrl 到 users 集合

点击昵称输入框 → type="nickname" 触发
  ↓
弹出系统键盘，顶部显示「使用微信昵称」建议
  ↓
用户确认后自动填充 → 防抖 800ms → 保存

### 再次进入

app.js onLaunch → auth.getUserInfo() 读取本地缓存
  ↓
有数据 → 自动恢复登录态
无数据 → 显示「微信一键登录」按钮

## 四、文件清单

| 文件 | 作用 |
|------|------|
| cloudfunctions/login/index.js | 云函数：获取 OPENID + CRUD users 集合 |
| miniprogram/utils/auth.js | 前端登录工具：silentLogin / saveProfile / getUserInfo / clearUserInfo |
| miniprogram/app.js | 启动时恢复登录态 |
| miniprogram/pages/settings/settings.js | 登录页逻辑 |
| miniprogram/pages/settings/settings.wxml | 登录页 UI |
| miniprogram/pages/settings/settings.wxss | 登录页样式 |

## 五、错误处理

| 场景 | 表现 | 处理方式 |
|------|------|----------|
| wx.login 失败 | 控制台 [auth] wx.login 失败 | Toast 提示，按钮恢复可点击 |
| 云函数调用失败 | 控制台 [auth] 云函数调用失败 | Toast 提示「登录失败」 |
| 用户拒绝选择头像 | 无回调（不触发 chooseavatar） | 无操作，保持原头像 |
| 头像上传失败 | 控制台 [auth] 头像上传失败 | 仅保存昵称，头像留空 |
| 云函数数据库异常 | 云函数内 try-catch 兜底 | 仍正常返回 OPENID |
| users 集合未创建 | 云函数内 try-catch 兜底 | 返回 OPENID，不影响登录 |

## 六、扩展：在其他页面获取登录态

方式 A：从 app 全局变量读取
  var app = getApp()
  if (app.globalData.userInfo) { /* 已登录 */ }

方式 B：直接从本地缓存读取
  var auth = require('../../utils/auth')
  var user = auth.getUserInfo()
  if (user) { /* 已登录 */ }
